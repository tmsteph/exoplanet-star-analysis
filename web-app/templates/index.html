Exoplanet Star Analysis

In this Project, We used the Nasa Exoplanet archive, as well as the PHL Habitable Exoplanet Catalog, in order to use machine learning to Identify habitable planets. Using this data, we analyze the stars these planets orbit around in order to narrow down our search for life!

The first confirmed discovery of an exoplanet occured in 1992. Since then, the number has doubled every 27 months! Machine learning will be neccesary in the future as the rate of information gathering continues to outpaces the ability of humans to process manually. We plan to help!
