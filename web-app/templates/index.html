XGBoost

After trying all the standard methods of machine learning, we came across XGboost.

This library is an implimentation of "gradiant boosted decision trees". This means that it uses an ensemble of smaller decision trees in series, each enhancing the final model.

What is special about XGBoost is the speed without losing accuracy! XGboost employs regularization, split Finding, and system design techniques to become one of the most popular libraries used by data scientists today.
